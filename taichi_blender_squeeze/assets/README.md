# Assets

This folder is intentionally empty.

The first version of the Blender scene is procedural: it creates the squishy object material, acrylic plates, floor, lighting, and camera from code. Optional upgrades can be placed here later:

- HDRI environment maps for richer reflections.
- Brushed acrylic or studio floor textures.
- Decorative base, labels, or product-shot props.

Reference repositories:

- `Tbl0x7D6/ACG-Project` is MIT licensed and includes Blender/render assets such as HDRI/material folders.
- `jason-huang03/SPH_Project` demonstrates a particle-to-surface-to-Blender workflow, but assets should still be checked before copying.

Do not copy third-party assets into this folder until their license and attribution requirements are confirmed.
